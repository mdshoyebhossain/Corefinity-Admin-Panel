## 📖 About

**Corefinity Admin Panel** is a modern, enterprise-grade control panel designed for managed hosting platforms. It provides a complete solution for managing deployments, environments, support tickets, user profiles, and system settings with a beautiful, responsive interface that works on all devices.

## ✨ Key Features

### 🎨 Modern UI/UX
- Clean, professional design with smooth animations
- Dark/Light mode toggle with persistent storage
- Fully responsive layout (mobile, tablet, desktop)
- 40+ UI components and patterns

### 📊 Dashboard Analytics
- Real-time statistics with Chart.js visualizations
- Deployment trends and environment distribution
- Recent activity feeds
- Key performance metrics

### 🚀 Deployment Management
- Complete deployment history with status tracking
- Rollback and log viewing capabilities
- Pipeline configuration management
- Repository integration forms with validation

### ☁️ Environment Monitoring
- Kubernetes cluster overview
- Pod, node, and resource monitoring
- Firewall rule management
- Pipeline execution status

### 🎫 Support Ticket System
- Full ticket lifecycle management
- Customer vs internal note distinction
- Ticket history tracking
- Priority and status filtering

### 👤 User Profile Management
- SSH key management
- API token generation and revocation
- Two-factor authentication
- IP whitelist configuration

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, CSS Variables)
- JavaScript ES6+
- Chart.js for data visualization
- Font Awesome 6 for icons
- Google Fonts (Inter)
